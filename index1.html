Projects

Project 1

This project is a classic Rock-Paper-Scissors game developed using HTML, CSS, and JavaScript. The game allows users to play against the computer, which randomly selects Rock, Paper, or Scissors. The user can select their choice, and the game determines the winner based on the traditional rules: Rock beats Scissors, Scissors beats Paper, and Paper beats Rock. The game includes a score tracker and a simple, user-friendly interface with animations and sound effects.

Project 2

This project is a Tic-Tac-Toe game built with HTML, CSS, and JavaScript. The game features a 3x3 grid where two players take turns to mark X and O on the grid. The game automatically checks for win conditions, such as three marks in a row, column, or diagonal. It also includes a restart button to reset the game and start over. The design is clean and minimalist, with a focus on providing a smooth and engaging user experience.
